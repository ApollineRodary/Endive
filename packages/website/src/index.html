Endive

A pedagogical proof assistant

Endive aims to be a pedagogical proof assistant inspired by Coq and Lean but providing a smoother learning curve. The current version of the project is accessible online, fell free to test it and give us a feedback :)

Endive is a proof assistant for the Integrated Project computer science course at ENS de Lyon.

Learning Endive is easy. You can start by reading the tutorial which is a good introduction to the language and the proof assistant.
